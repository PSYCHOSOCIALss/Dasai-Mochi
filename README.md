> ใช้บอร์ด Arduino ในการควบคุมการทำงานของระบบทั้งหมด

> แสดงผลผ่านจอแสดงผล tft 0.96 inch เพื่อแสดงข้อความหรือสถานะการทำงาน

> ระบบสามารถตอบสนองต่ออินพุตจากปุ่มกดหรือเซ็นเซอร์ได้แบบ Real-time

> ใช้โครงสร้างโปรแกรมแบบ Loop และ Function เพื่อจัดการลำดับการทำงานอย่างเป็นระบบ

> สามารถกำหนดรูปแบบการแสดงผลหรือแอนิเมชันได้ผ่านการเขียนโค้ด

> รองรับการปรับแต่งพฤติกรรมของอุปกรณ์ผ่านการแก้ไขค่าตัวแปรภายในโปรแกรม

> ระบบทำงานต่อเนื่องอัตโนมัติเมื่อมีการจ่ายไฟเข้า Arduino
